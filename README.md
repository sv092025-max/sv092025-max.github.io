
# Portfolio - Samuel Vasquez

This is a Neon Dark portfolio that showcases mini-store demos created with HTML, CSS, and JavaScript.
The `mini-stores/demo1` folder contains the uploaded mini-store project (index.html, css, js, images).

## How to publish
1. Push this folder to a GitHub repository.
2. In the repo settings, enable GitHub Pages (branch: main, folder: /root).
3. Access the live site at: https://sv092025-max.github.io/
